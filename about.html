About this project

Makeup shade names are famously unhelpful. "Velvet Plum," "Midnight Berry," "Spiced Rosewood." While evocative, they make it nearly impossible to find what you're actually looking for across brands.

So I built a color-matching tool that skips the names entirely. Pick a color on the wheel and it finds lip products that genuinely match, ranked by how close they are perceptually using the same standard (CIELAB and Delta E) that cosmetics manufacturers use internally.

To make that possible, I first had to build the database: automated color extraction for 9,000+ products using image classification, semantic segmentation, and perceptual color clustering, turning unstructured product photos into a searchable color index.
